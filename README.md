# 泽鳌飞兴水产园区智慧平台 - 改进版

## 项目概述
这是一个响应式智慧水产养殖大屏可视化系统，专为现代化水产养殖园区设计。

## 主要改进

### 1. 响应式设计 ✅
- 完全响应式布局，支持从手机到超大屏幕的所有设备尺寸
- 采用CSS Grid和Flexbox实现弹性布局
- 使用clamp()函数实现流体字体大小
- 支持横屏和竖屏自适应
- 针对触摸设备优化交互区域

**支持的屏幕尺寸：**
- 超大屏幕 (>= 2560px)
- 大屏幕 (1920px - 2559px)
- 标准屏幕 (1440px - 1919px)
- 笔记本 (1280px - 1439px)
- 平板横屏 (1024px - 1279px)
- 平板竖屏 (768px - 1023px)
- 手机横屏 (640px - 767px)
- 手机竖屏 (< 640px)

### 2. 监控点优化 ✅
- 地图上均匀分布**52个监控点**
- 监控点类型包括：
  - 水质监测（绿色）
  - 溶氧监测（青色）
  - 温度监测（蓝色）
  - 视频监控（黄色）
  - 告警点位（橙色）
- 支持在线/离线状态显示
- 可点击查看详细信息
- 支持筛选功能（全部/在线/离线）
- 实时统计监控点数量

### 3. 水产养殖关键指标（条形图展示）✅
替换了原来的养殖池信息板块，改为使用Chart.js展示：
- 存活率
- 生长速度
- 饲料转化
- 水质指数
- 溶氧水平
- 疾病控制

**特点：**
- 横向条形图，清晰直观
- 彩色编码，便于识别
- 悬停显示详细数值
- 响应式设计

### 4. 气象监测图表化 ✅
将原来的图标展示改为专业图表：
- 24小时温度和湿度趋势双轴折线图
- 实时数据摘要（温度、湿度、风速、气压）
- 平滑曲线展示
- 双Y轴设计，便于对比

### 5. 设备控制塘区选择功能 ✅
- 新增塘区选择下拉菜单
- 支持5个塘区切换：
  - 一号塘区
  - 二号塘区
  - 三号塘区
  - 四号塘区
  - 五号塘区
- 显示每个塘区的：
  - 在线/离线状态
  - 设备总数
  - 运行中设备数
- 切换时有动画效果

## 使用方法

### 1. 直接使用
双击 `index.html` 文件即可在浏览器中打开。

### 2. 本地服务器
推荐使用本地服务器运行以获得最佳体验：

```bash
# 使用Python 3
python -m http.server 8000

# 使用Node.js
npx serve

# 使用VS Code Live Server插件
右键点击 index.html -> Open with Live Server
```

然后在浏览器中访问：`http://localhost:8000`

## 功能说明

### 地图监控
- **刷新按钮**：重新生成监控点
- **筛选按钮**：
  - 全部：显示所有监控点
  - 在线：仅显示在线监控点
  - 离线：仅显示离线监控点
- **点击监控点**：查看详细信息

### 设备控制
- **选择塘区**：从下拉菜单选择不同塘区
- **设备开关**：点击设备项切换开关状态
- 支持的设备：
  - 投饵机
  - 充氧系统
  - 增氧机
  - 水泵
  - 照明
  - 警戒系统

### 视频监控
- 点击视频框可上传自定义图片
- 支持4路视频监控显示

### 数据展示
- 产量排行榜
- 水产养殖关键指标（条形图）
- 气象监测（折线图）
- 水质监测（实时数据）
- 流量统计（圆形进度）

## 自定义配置

### 修改颜色主题
编辑 `css/style.css` 中的CSS变量：

```css
:root {
    --bg-primary: #0a1628;
    --accent-cyan: #00d4ff;
    --accent-green: #00ff88;
    /* 更多颜色... */
}
```

### 调整监控点数量
修改 `js/main.js` 中的配置：

```javascript
const totalMarkers = 52; // 修改此值
const cols = 8;          // 每行点数
```

### 修改塘区数据
修改 `js/main.js` 中的 `pondData` 对象：

```javascript
let pondData = {
    pond1: { name: '一号塘区', devices: 12, running: 8, status: 'online' },
    // 添加或修改塘区...
};
```

## 数据更新
